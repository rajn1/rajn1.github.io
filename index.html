The Maze Game

A Fundamentals of Computer Science II project built in Java with Kruskal's Algorithm to procedurally generate mazes up to 100 x 60 blocks. Players can solve puzzles manually or hand control to automated DFS/BFS solvers. Created with Jonathan Northcott.

Instructions:

  • Use h and arrow keys to select start/end points.
  • Press s to form the maze step-by-step, i to skip animation.
  • r creates a new maze, while x restarts the current puzzle (press twice to quit).
  • n cycles color palettes and p enables player-controlled solving.
  • d runs depth-first search; b runs breadth-first search.

Download and Play Here

Forbidden Island Game

A CS II survival challenge where players crash-land on an island, mark safe land, and locate a helicopter before flood waters or roaming enemies end the run. Built with Jonathan Northcott.

Instructions:

  • Press h to start and use WASD to move.
  • f reveals obstacles, m pauses enemies, and n toggles helicopter visibility.
  • r randomizes the island and t toggles the full map view.
